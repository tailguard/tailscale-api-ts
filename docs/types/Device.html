A Tailscale device (sometimes referred to as node or machine), is any computer or mobile device that joins a tailnet.

Each device has a unique ID (nodeId in the schema below) that is used to identify the device in API calls. This ID can be found by going to the Machines page in the admin console, selecting the relevant device, then finding the ID in the Machine Details section. You can also list all devices in the tailnet to get their nodeId values.

      • Optionalpcp?: boolean

        'true' if PCP port-mapping service exists on your router.

      • Optionalpmp?: boolean

        'true' if NAT-PMP port-mapping service exists on your router.

      • Optionaludp?: boolean

        'true' if UDP traffic is enabled on the current network; if 'false', Tailscale may be unable to make direct connections, and will rely on our DERP servers.

      • Optionalupnp?: boolean

        'true' if UPnP port-mapping service exists on your router.

    • Optionalendpoints?: string[]
